## 로드맵은 어떻게 만들어졌을까요?

>로드맵은 [Balsamiq](https://balsamiq.com/download/)을 이용해 만들어졌습니다.

* 저장소를 클론하세요
* Balsamiq을 열고 <kbd>Project > Import > Mockup JSON</kbd>, 순으로 클릭합니다.
* 업데이트 하고 싶은 로드맵의 JSON을 복사, 붙여넣기 합니다.
* 변경 사항을 추가합니다.
* Export the JSON file <kbd>Project > Export > Mockup to JSON</kbd> 순으로 진행하고 <kbd>src</kbd> 폴더에 넣습니다.
* Export the image <kbd>Project > Export > Current Mockup to PNG</kbd> 순으로 진행하고 <kbd>images</kbd> 폴더에 넣습니다.
* 변경사항을 커밋하고 PR합니다.
